## 1.0.0

- Initial release
- `PureRouteObserver` - Navigator observer for tracking route changes
- `PureRouteAwareMixin` - Mixin for route visibility callbacks without rebuilds
- `PureRouteRegistry` - Internal registry for route-widget mappings
- Automatic filtering of popup routes (dialogs, bottom sheets)
- `isRouteVisible` property for checking current visibility state
